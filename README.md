# Cartão de Perfil Responsivo

Projeto de cartão de perfil desenvolvido com HTML e CSS, com foco em layout responsivo, estilização moderna e boas práticas de front-end.

---

## Tecnologias utilizadas

- HTML5
- CSS3

---

##  Funcionalidades

- Layout centralizado
- Design moderno com gradiente
- Animações com hover
- Responsividade para diferentes telas
- Links para redes sociais
- Email clicável

---

## Aprendizados

Neste projeto pratiquei:

- Estruturação com HTML
- Estilização com CSS
- Flexbox
- Responsividade
- Animações com `hover` e `transition`
